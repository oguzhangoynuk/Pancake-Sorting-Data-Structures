# Pancake Sorting with Doubly Linked List 🥞

This project implements the **Pancake Sorting** algorithm, not on a standard array, but on a **Doubly Linked List**. This approach demonstrates advanced pointer manipulation and a deep understanding of non-contiguous data structures.

## 🧠 The Algorithm
Pancake sorting is a variation of the sorting problem where the only allowed operation is reversing the elements from the start to a given index (a "flip"). 

### Technical Implementation:
* **Data Structure:** Doubly Linked List (prev/next pointers).
* **Flip Operation:** Instead of swapping values, this implementation rearranges the **pointers** themselves, which is a more advanced memory management task.
* **FindMax:** Scans the unsorted portion of the list to find the largest "pancake" to move it to its correct position.
